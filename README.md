## Getting Started
1. Clone the repository:
```bash
git clone https://github.com/your-username/instaclone.git
```
2. Install dependencies:
```bash
cd instaclone
npm install
```
3. Start the project:
```bash
npx expo start
```
4. Open it on your device using Expo Go.
